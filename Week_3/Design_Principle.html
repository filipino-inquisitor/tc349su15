Design Principles

Rule of Thirds (Appeal):

A technique in composition in which a medium is divided into thirds, creating aesthetic positions for design.

The way Fitbit has their page layout is in horizontal thirds. The user is able to clearly distinct seperate areas by a clean horizontal change.

80/20 Rule (Usability):

A high percentage of effects in any large system are caused by a low percentage of variables.

Mobile applications today use this rule to the fullest. Developers want to maximize the overall functionality by giving the users the basics. With a simplest sign-on process as well as navigation throughout the application, USAA gets an outstanding on using the 80/20 rule flawlessly.

Hick's Law (Usability):

The time it takes to make a decision increases as the number of alternatives increases.

There are a lot of options for the user to select once they enter Amazon's homepage. With so many choices, it increases the time the user takes on making their decision. They offer so much that it sometimes hurts the efficency of the overall masses.

Signal-to-Noise Ratio (Learning):

The ratio of relevant to irrelevant information in a display.

Extremely simple. There is only the search box which is what users use the brand for. Google only shows what is relevant as well to most searches.

Ockham's Razor (Decision):

Given the choice between functionally equivalent designs, the simplest design should be selected.

Similiar to the S/N Ratio, it displays a clean aspect with not many irrelevant information on the page. It shows images and those images can be selected. This gives the website both a functionality and designability that truly meets clean expectations.
